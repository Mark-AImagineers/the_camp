# The Camp

> A post-apocalyptic shelter management game where you plan, then lose control — and live with what comes back.

---

## What is this

The Camp is a browser-based management/narrative game. You run a survivor camp
in a collapsed world. You assign roles, plan expeditions, equip your people —
then you deploy them and lose all control. The run plays out as a narrative feed.
You wait. You find out what happened.

The emotional hook is not loot. It is people.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React + Vite + Tailwind |
| Backend | FastAPI |
| Database | PostgreSQL |
| Cache | Redis |
| Streaming | SSE (expedition feed) |
| Narrative | Claude API |
| Desktop | Electron (Steam distribution) |
